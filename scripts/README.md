# 📂 Scripts

Carpeta para scripts de utilidad del proyecto.

## Scripts disponibles

### `bootstrap.ps1` — Iniciar proyecto nuevo
```powershell
.\scripts\bootstrap.ps1 "NombreDelProyecto"
# o con ruta personalizada:
.\scripts\bootstrap.ps1 "NombreDelProyecto" "C:\ruta\destino"
```
Crea la estructura del proyecto, copia `.mission_control/` y abre CONTEXTO.md.
