# Design System: Equilibrium Chonchi

## Colores
- `primary`: `#025273` (Botones, links, títulos principales, iconos de servicios)
- `primaryDark`: `#0b5074` (Hover de botones, variaciones UI)
- `accent`: `#e20514` (CTA de agendamiento, urgencias)
- `bgPrimary`: `#ffffff` (Fondo general limpio y clínico)
- `bgSecondary`: `#f8fafc` (Fondos sutiles alternativos, asumiendo tono neutro)
- `textPrimary`: `#1c1c1c` (Texto general, gran contraste)
- `textSecondary`: `#4b5563` (Subtítulos, descripciones secundarias)

## Tipografía
- **Heading**: `Plus Jakarta Sans`, sans-serif (Apariencia amable pero moderna y profesional)
- **Body**: `Inter`, sans-serif (Alta legibilidad)

## Componentes UI
- Botones redondeados suaves (`border-radius` moderado), sombras sutiles (Premium/Elegante).
- Iconografía clara de Lucide React (ej. `heart-pulse`, `user`, `stethoscope` equivalente).
